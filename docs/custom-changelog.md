# Firefly 定制功能记录

本文记录 `Embersinthewind/Firefly` 相对上游 [CuteLeaf/Firefly](https://github.com/CuteLeaf/Firefly) 的主要功能变化。它是功能概览，不代替 Git 提交历史和正式版本号。

## 个人展示与导航

- 首页改为个人项目展示，支持 GitHub 仓库卡片和能力矩阵。
- 顶部导航改为悬浮胶囊式导航，并整理为主页、文章、网站导航、动态、记录和关于。
- 新增 `/navigation/` 网站导航页，导航数据可通过作者接口提交到 GitHub。
- 首页分类快捷栏和信息组件按页面用途精简。

## 文章系统

- 新增 `/articles/` 分页文章库、分类筛选和封面式文章卡片。
- 写作轨迹移入文章列表左栏，以紧凑年度月历展示发布活跃度。
- 文章列表支持横向列表与三列卡片两种查看方式，并记住浏览器选择。
- 文章列表、分类和详情页采用专注内容的简化布局，归档入口从文章导航移除。
- 修复分页跳转、文章索引对齐和详情目录布局。

## 写作工作台

- 新增 `/write/` Markdown 写作页面。
- 支持导入与下载 Markdown、浏览器草稿、预览和 GitHub 发布。
- 支持文章摘要、标签、分类、发布日期、草稿、置顶和封面。
- 支持粘贴、拖入和选择正文图片，上传后实时预览并调整显示宽度。
- 支持封面上传预览、进度状态、点击更换和悬停删除。
- 文章 Slug 自动生成，浏览量不由作者手动填写。

## K-Vault 集成

- 在“站点设置 → 写作设置”中保存 K-Vault 地址、API Token、存储源、目录和链接模式。
- 写作配置仅保存在当前浏览器，不进入 GitHub。
- Firefly Worker 代理 K-Vault 上传，减少浏览器跨域问题。
- K-Vault 地址限制为公网 HTTPS，图片大小限制为 20 MiB。

## 安全作者代理

- 新增 Cloudflare Worker `/api/author/*` 作者接口。
- GitHub Token 只保存在 Cloudflare Secret，不下发浏览器。
- 使用作者密码和签名 `HttpOnly` Cookie 登录，默认保持 30 天。
- 限制允许编辑的配置文件和文章目录，并进行同源写入校验。
- GitHub API 错误会转换为更明确的发布提示。

## 其他页面

- 新增可视化站点配置和个人资料编辑。
- 新增分类式项目展示。
- 新增沉浸式 `/music/` 音乐页面。
- 主页横幅、状态组件和移动端导航按个人站点方向调整。
- 首页状态组件支持“上班 / 下班 / 休息”三种状态；上下班使用独立 GIF，休息继续使用个人头像。
- 修复 Swup 页面切换后首页侧栏残留或无法恢复的问题，专注内容页与首页会同步切换布局。

## 文档维护规则

以下变化应同步更新文档：

1. Cloudflare 变量或 Secret 名称发生变化。
2. GitHub 权限、作者登录或 Cookie 策略发生变化。
3. K-Vault API、存储源字段或浏览器保存方式发生变化。
4. 新增或移除公开页面、作者入口或数据文件。
5. 构建命令、Node.js 版本或部署方式发生变化。

## 壁纸组管理

- 默认 6 张桌面壁纸已归入 `默认壁纸` 文件夹。
- 默认 6 张移动端壁纸也已归入移动端的 `默认壁纸` 文件夹。
- 桌面端和移动端均支持选择单张壁纸或整个文件夹作为随机轮播壁纸组。
- 作者模式支持新建并命名壁纸组、批量上传、向现有组添加以及同格式替换单张壁纸。
- 壁纸缩略图支持右上角删除，并可通过拖拽调整轮播顺序。
- 壁纸文件通过 Cloudflare 作者代理写入 GitHub，不会修改访问者或作者电脑的本地目录。
